## O que eu faço

- **Desenvolvimento Web:** Criação de aplicações completas, da modelagem arquitetural do banco de dados à interface final responsiva e dinâmica.
- **Back-end & APIs:** Construção de serviços robustos utilizando PHP (Laravel) e Node.js (NestJS), com foco em separação de responsabilidades, filas e microsserviços.
- **Desktop & Baixo Nível:** Experiência prática com C++ moderno e C# (.NET), garantindo alto desempenho, controle de recursos e desenvolvimento a nível de sistema (Windows).
- **Engenharia & Qualidade:** Código escrito pensando em testabilidade. Uso rigoroso de TDD, testes unitários/integração, Clean Code e padrões de projeto para garantir entregas seguras e sustentáveis.
